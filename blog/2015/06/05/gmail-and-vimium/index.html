Gmail & Vimium

Jun 5th, 2015

Recently I memorized most of the GMail keyboard short cuts, but I was very disappointed with the fact that I could not quickly and easily open links. Using the Vimium extension, now I can.

1. Download Vimium

Chrome

Firefox

Not sure if browsers have it.

2. Stop it from stealing input

In order for the Gmail shortcuts to work you need to disable the conflicting Vimium ones. Open the Vimium configuration menu and add an exception for Gmail. I use the following:

  • Pattern - https?://mail.google.com/*
  • Keys - ? / g r y v x f u d o O n p

For GMail, all I care about is the shift + f key which opens links in a new window.

3. Disable focus stealing

Sites that automatically focus a form item will disable Vimium, until you exit the form item. This is annoying, but luckily Vimium can prevent that.

Open the Advanced options and click “Don’t let pages steal the focus on load”

4. Use Vimium

Some of the most basic commands I find useful are:

  • ? - Opens the Vimium help menu
  • f - Marks each link with a key-cord. Pressing the cord will open the link
  • F - Same as “f” but in a new windows
  • yy - Copy current URL

A lot of the other commands have a standard shortcut, so I generally ignore them.
