Introduction
I converted a Rails app I was working on just because I could.  I have a lot of experience with MySQL; specifically working around a lot of idiosyncrasies.  In reading, it appeared that PostgreSQL was better in many ways and since rails supports it out of the box it should just work.

Installing PostgreSQL
I use Mac OS X and they already have an installer here.  It did everything that was required to get PostgreSQL working.

Setting up PostgreSQL
Once PostgreSQL is up and running you have to install a user.  I used the supplied pgAdmin III.app to create a user, and three rails databases.
